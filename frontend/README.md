# Frontend - Cosmetic Management System

## Tech Stack
- Vue 3
- TypeScript
- Vite
- Pinia
- Vue Router
- Element Plus
- Axios

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

App will be available at `http://localhost:5173`

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```
